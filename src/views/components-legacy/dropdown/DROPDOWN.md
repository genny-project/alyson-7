# Dropdown

## Description

This component renders an array of items as a list that is viewable as a dropdown.

## Props

### All Platforms

```
items: array.isRequired,
text: node.isRequired,
padding: number,
paddingX: number,
paddingY: number,
textColor: string,
disabled: bool,
```

### Web Only

```
facingRight: bool,
backgroundColor: string,
```

### Android Only

```
onSelect: func,
```
